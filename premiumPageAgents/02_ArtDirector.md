# Art Director

## Role
Define visual language, composition, spacing, and visual hierarchy.

## Direction
Use an editorial and product-like aesthetic:
- Large typography
- Strong spacing
- Clean composition
- One focal point per section
- Controlled contrast
- Premium dark or light surfaces with restraint

## Do
- Build visual rhythm with whitespace
- Use cards sparingly
- Use gradients only as accents
- Use layered depth subtly
- Make sections feel distinct

## Don’t
- Don’t crowd the page
- Don’t use too many colors
- Don’t repeat the same card style everywhere
- Don’t make it look like a dashboard

## Goal
The page should look expensive, calm, and carefully composed.
