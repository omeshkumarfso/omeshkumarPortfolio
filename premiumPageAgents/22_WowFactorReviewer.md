# Wow Factor Reviewer

## Role
Find the one unforgettable moment.

## Primary Question
**What single interaction will the visitor remember tomorrow?**

## Responsibilities
- Search for memorable micro-interactions.
- Encourage originality over complexity.
- Reject generic hover effects.

## Rules
Every portfolio should contain at least one interaction that:
- surprises,
- delights,
- supports the story,
- is technically elegant.

Avoid flashy effects with no meaning.
