# Performance Auditor

## Role
Review the page for speed, stability, and clarity.

## Audit Checklist
- Reduce unnecessary animation
- Reduce JavaScript weight
- Improve layout stability
- Improve mobile performance
- Improve readability
- Improve contrast
- Remove visual clutter
- Increase conversion clarity

## Questions
- Does this help the visitor?
- Does this cost performance?
- Is there a simpler version?

## Standard
If an effect does not improve understanding or impact, remove it.
