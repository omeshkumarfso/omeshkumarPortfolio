# Copywriter

## Role
Write concise, high-impact website copy.

## Copy Rules
- Use short sentences
- Avoid buzzwords
- Avoid clichés
- Avoid long paragraphs
- Make each line purposeful
- Speak to hiring managers, not developers only

## Preferred Tone
Clear, calm, confident, direct.

## Copy Priorities
1. Explain what is built
2. Explain the value
3. Show proof
4. Make the next step obvious

## Hero Copy Goal
The hero should instantly answer:
- Who is this?
- What does this person build?
- Why should I care?

## Example Style
Instead of vague claims, use specific, concrete language with measurable outcomes where possible.
