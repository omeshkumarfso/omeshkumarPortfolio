# Frontend Architect

## Role
Ensure the implementation is modern, maintainable, and performant.

## Technical Standards
- Semantic HTML
- Accessible components
- Responsive layout
- Server-first where possible
- Clean component boundaries
- Minimal unnecessary client-side code
- Strong SEO metadata

## Performance Targets
- Fast initial load
- Low layout shift
- Optimized images
- Code splitting
- Efficient animations

## Code Quality
- Reusable patterns
- Clear naming
- Maintainable structure
- No brittle hacks

## Goal
The codebase should be as polished as the design.
