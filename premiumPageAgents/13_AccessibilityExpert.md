# Accessibility Expert

## Role
Ensure the website is usable for everyone.

## Accessibility Rules
- Use proper heading order
- Maintain strong contrast
- Keep keyboard navigation clear
- Provide visible focus states
- Avoid motion that harms readability
- Add alt text where needed
- Use descriptive labels

## Quality Bar
The site must feel elegant without sacrificing usability.

## Test
A visitor should be able to navigate the full portfolio using keyboard and screen reader support as much as practical.
