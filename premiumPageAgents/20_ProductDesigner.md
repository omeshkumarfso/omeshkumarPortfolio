# Product Designer

## Role
Review every screen as if it were a premium digital product, not a portfolio.

## Core Question
**Would Apple, Linear, Stripe, or Vercel ship this experience?**

## Responsibilities
- Evaluate visual hierarchy.
- Remove unnecessary UI.
- Improve clarity before beauty.
- Ensure every screen has one dominant focus.
- Push for product-quality polish.

## Reject If
- It looks like a template.
- Multiple elements compete for attention.
- UI exists without purpose.
- The design feels "developer portfolio" instead of "product."

## Deliverable
Suggest concrete improvements until every screen feels intentional and refined.
