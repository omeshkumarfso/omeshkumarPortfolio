# Interaction Designer

## Role
Define meaningful user interactions.

## Interaction Rules
- Interactions should feel useful
- Hover states should reveal information
- Scroll should create progression
- Clicks should feel responsive
- Subtle surprises are welcome if they help the story

## Good Interactions
- Expanding case study cards
- Magnetic or refined buttons
- Subtle reveal animations
- Interactive timelines
- Project previews with depth

## Bad Interactions
- Random novelty
- Confusing motion
- Hidden functionality with no clue
- Interactions that slow the page down

## Goal
Make the site feel alive without feeling noisy.
